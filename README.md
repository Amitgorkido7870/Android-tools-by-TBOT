# Termux Tools Installer
![tbot](https://github.com/TecnicalBot/tbot/blob/main/images/tbot.png)


### Installation
```bash
pkg update
pkg install wget -y
wget -O $PREFIX/bin/tbot https://raw.githubusercontent.com/TecnicalBot/tbot/main/tbot.sh
chmod +x $PREFIX/bin/tbot
```

### Launch Installer
```bash
tbot
```
